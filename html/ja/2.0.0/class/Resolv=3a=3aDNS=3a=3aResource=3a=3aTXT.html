class Resolv::DNS::Resource::TXT

クラスの継承リスト: Resolv::DNS::Resource::TXT < Resolv::DNS::Resource < Resolv::DNS::Query < Object < Kernel < BasicObject

要約

DNS リソースの TXT レコード を表す抽象クラスです。

Resolv::DNS#getresource で TXT レコードを得たい場合は Resolv::DNS::Resource::IN::TXT を使うべきです。
