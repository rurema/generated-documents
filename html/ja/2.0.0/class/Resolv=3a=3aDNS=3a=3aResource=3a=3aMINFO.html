class Resolv::DNS::Resource::MINFO

クラスの継承リスト: Resolv::DNS::Resource::MINFO < Resolv::DNS::Resource < Resolv::DNS::Query < Object < Kernel < BasicObject

要約

DNS リソースの MINFO レコード を表す抽象クラスです。

Resolv::DNS#getresource で MINFO レコードを得たい場合は Resolv::DNS::Resource::IN::MINFO を使うべきです。
