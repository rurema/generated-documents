constant Socket::SO_PRIORITY

SO_PRIORITY -> Integer[permalink][rdoc]

The protocol-defined priority for all packets on this socket。 BasicSocket#getsockopt, BasicSocket#setsockopt の第2引数(optname)に使用します。
