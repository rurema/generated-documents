instance method Thread::Backtrace::Location#label

label -> String[permalink][rdoc]

self が表すフレームのラベルを返します。通常、メソッド名、クラス名、モ ジュール名などで構成されます。

例: Thread::Backtrace::Location の例1を用いた例 

loc = c(0..1).first
loc.label # => "a"
