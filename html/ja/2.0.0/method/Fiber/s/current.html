singleton method Fiber.current

current -> Fiber[permalink][rdoc] [added by fiber]

このメソッドが評価されたコンテキストにおける Fiber のインスタンスを返します。

例: 

fr = Fiber.new do
  Fiber.current
end

fb = fr.resume
p fb.equal?(fr) # => true

p Fiber.current # => #<Fiber:0x91345e4>
p Fiber.current # => #<Fiber:0x91345e4>
