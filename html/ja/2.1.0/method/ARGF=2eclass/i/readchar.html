instance method ARGF.class#readchar

readchar -> Integer[permalink][rdoc]

ARGFから 1 文字読み込んで、その文字に対応する Fixnum を返します。EOF に 到達した時には EOFErrorを発生します。

[EXCEPTION] EOFError:
EOFに達した時発生する 
$ echo "foo" > file
$ ruby argf.rb file

ARGF.readchar  # => "f"
ARGF.readchar  # => "o"
ARGF.readchar  # => "o"
ARGF.readchar  # => "\n"
ARGF.readchar  # => end of file reached (EOFError)
