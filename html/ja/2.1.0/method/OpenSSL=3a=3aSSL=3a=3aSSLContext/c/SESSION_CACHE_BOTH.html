constant OpenSSL::SSL::SSLContext::SESSION_CACHE_BOTH

SESSION_CACHE_BOTH -> Integer[permalink][rdoc]

サーバ側、クライアント側両方でセッションをキャッシュすることを意味します。

OpenSSL::SSL::SSLContext#session_cache_mode= に 渡すフラグとして用います。

実際には OpenSSL::SSL::SSLContext::SESSION_CACHE_SERVER と OpenSSL::SSL::SSLContext::SESSION_CACHE_CLIENT のビット論理和 を取った値です。
