library win32/resolv

要約

win32/resolv は Win32 プラットフォームで名前解決に関する情報を取得する ためのライブラリです。

クラス

Win32::Resolv

名前解決に関する情報を取得するためのクラスです。

同時にrequireされるライブラリ

win32/registry

win32/registry は Win32 プラットフォームでレジストリをアクセスするための ライブラリです。Win32 API の呼び出しに Win32API を使います。
