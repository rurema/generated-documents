instance method Dir#pos=

pos=(pos)[permalink][rdoc]
seek(pos) -> self

ディレクトリストリームの読み込み位置を pos に移動させます。 pos は Dir#tell で与えられた値でなければなりま せん。

[PARAM] pos:
変更したい位置を整数で与えます。
[EXCEPTION] IOError:
既に自身が close している場合に発生します。

例: 

Dir.open("testdir") do |d|
  d.read                   # => "."
  i = d.tell               # => 12
  d.read                   # => ".."
  d.seek(i)                # => #<Dir:0x401b3c40>
  d.read                   # => ".."
end
