singleton method Dir.delete

delete(path) -> 0[permalink][rdoc]
rmdir(path) -> 0
unlink(path) -> 0

ディレクトリを削除します。ディレクトリは空でなければいけませ ん。ディレクトリの削除に成功すれば 0 を返します。

[PARAM] path:
ディレクトリのパスを文字列で指定します。
[EXCEPTION] Errno::EXXX:
失敗した場合に発生します。

例: 

Dir.delete("/tmp/hoge-jbrYBh.tmp")
