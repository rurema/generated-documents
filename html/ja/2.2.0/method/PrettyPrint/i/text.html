instance method PrettyPrint#text

text(obj) -> ()[permalink][rdoc]
text(obj, width = obj.length) -> ()

obj を width カラムのテキストとして自身に追加します。

[PARAM] obj:
自身に追加するテキストを文字列で指定します。
[PARAM] width:
obj のカラムを指定します。指定されなかった場合、obj.length が利用されます。
