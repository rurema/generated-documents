instance method Range#size

size -> Integer | Float::INFINITY | nil[permalink][rdoc]

範囲内の要素数を返します。始端、終端のいずれかのオブジェクトが Numeric のサブクラスのオブジェクトではない場合には nil を返します。

例: 

(10..20).size    # => 11
("a".."z").size  # => nil
(-Float::INFINITY..Float::INFINITY).size # => Infinity
