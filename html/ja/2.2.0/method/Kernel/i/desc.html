instance method Kernel#desc

desc(description) -> String[permalink][rdoc] [added by rake]

直後の Rake タスクの説明を登録します。

[PARAM] description:
直後のタスクの説明を指定します。

例: 

desc "Run the Unit Tests"
task :test => [:build] do
  runtests
end
