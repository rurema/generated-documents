instance method REXML::SAX2Listener#attlistdecl

attlistdecl(element, pairs, contents) -> ()[permalink][rdoc]

DTDの属性リスト宣言に対し呼び出されるコールバックメソッドです。

[PARAM] element:
要素名が文字列で渡されます
[PARAM] pairs:
属性名とそのデフォルト値の対応が { 属性名文字列 => デフォルト値文字列(無ければnil) } という ハッシュテーブルで渡されます
[PARAM] contents:
文書内の属性リスト宣言の文字列がそのまま渡されます
