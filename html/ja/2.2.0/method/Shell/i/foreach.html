instance method Shell#foreach

foreach(path = nil, &block) -> ()[permalink][rdoc]

pathがファイルなら, File#foreach pathがディレクトリなら, Dir#foreach の動作をします。

[PARAM] path:
ファイルもしくはディレクトリのパスを文字列で指定します。

使用例 

require 'shell'
Shell.verbose = false
sh = Shell.new
sh.foreach("/tmp"){|f|
  puts f
}
