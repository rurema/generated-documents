instance method JSON::Generator::GeneratorMethods::String#to_json

to_json -> String[permalink][rdoc]

自身から生成した JSON 形式の文字列を返します。

自身のエンコードは UTF-8 であるべきです。 "\u????" のように UTF-16 ビッグエンディアンでエンコードされた文字列を返すことがあります。
