instance method Net::HTTP#local_host=

local_host=(host)[permalink][rdoc]

接続に用いるローカルホスト名を指定します。

nil の場合システムが適当にローカルホストを 決めます。

デフォルトは nil です。

[PARAM] host:
ホスト名、もしくはアドレスを示す文字列 
例

require 'net/http'

http = Net::HTTP.new("www.example.com")
http.local_host = "192.168.0.5"
http.local_port = "53043"

http.start do |h|
  p h.get("/").body
end
