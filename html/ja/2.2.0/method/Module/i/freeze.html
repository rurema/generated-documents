instance method Module#freeze

freeze -> self[permalink][rdoc]

モジュールを凍結（内容の変更を禁止）します。

凍結したモジュールにメソッドの追加など何らかの変更を加えようとした場合 に RuntimeError が発生します。

[SEE_ALSO] Object#freeze

例: 

module Foo; end
Foo.freeze

module Foo
  def foo; end
end # => RuntimeError: can't modify frozen module
