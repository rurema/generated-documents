instance method SignalException#signo

signo -> Integer[permalink][rdoc]

self のシグナル番号を返します。

例: 

p Signal.signame(1) # => "HUP"
begin
  Process.kill('HUP',Process.pid)
  sleep
rescue SignalException => e
  p e.signo # => 1
end
