module function Math.#cos

cos(x) -> Float[permalink][rdoc]

x の余弦関数の値をラジアンで返します。

[PARAM] x:
実数
[RETURN]
[-1, 1] の実数
[EXCEPTION] TypeError:
x に数値以外を指定した場合に発生します。
[EXCEPTION] RangeError:
x に実数以外の数値を指定した場合に発生します。 
例

Math.cos(Math::PI) # => -1.0
