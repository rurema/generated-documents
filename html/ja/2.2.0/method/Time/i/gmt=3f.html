instance method Time#gmt?

gmt? -> bool[permalink][rdoc]
utc? -> bool

self のタイムゾーンが協定世界時に設定されていれば真を返します。 

t = Time.local(2017,9,19,15,0,0)   # => 2017-09-19 15:00:00 +0900
t.utc?                             # => false
utc_t = t.getutc                   # => 2017-09-19 06:00:00 UTC
utc_t.utc?                         # => true
