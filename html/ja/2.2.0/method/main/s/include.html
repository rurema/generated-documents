singleton method main.include

include(*modules) -> self[permalink][rdoc]

引数 modules で指定したモジュールを後ろから順番にインクルードします。

[PARAM] moddules:
Module のインスタンス( Enumerable など)を指定します。
[EXCEPTION] ArgumentError:
継承関係が循環してしまうような include を行った場 合に発生します。 
例:

include Math

hypot(3, 4)  # => 5.0
