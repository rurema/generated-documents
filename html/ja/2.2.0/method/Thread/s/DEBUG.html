singleton method Thread.DEBUG

DEBUG -> Integer[permalink][rdoc]

スレッドのデバッグレベルを返します。

スレッドのデバッグレベルが 0 のときはなにもしません。 それ以外の場合は、スレッドのデバッグログを標準出力に出力します。 初期値は 0 です。 使用するためには、THREAD_DEBUG を -1 にして Ruby をコンパイルする必要が あります。 

例

Thread.DEBUG # => 0
