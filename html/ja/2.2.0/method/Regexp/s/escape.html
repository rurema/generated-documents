singleton method Regexp.escape

escape(string) -> String[permalink][rdoc]
quote(string) -> String

string の中で正規表現において特別な意味を持つ文字の直前にエ スケープ文字(バックスラッシュ)を挿入した文字列を返します。

[PARAM] string:
正規表現において特別な意味をもつ文字をもつ文字列を指定します。 
rp = Regexp.escape("$bc^")
puts rp #=> \$bc\^
