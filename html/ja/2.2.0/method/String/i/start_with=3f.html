instance method String#start_with?

start_with?(*strs) -> bool[permalink][rdoc]

self の先頭が strs のいずれかであるとき true を返します。

[PARAM] strs:
パターンを表す文字列 (のリスト) 
例

"string".start_with?("str")          # => true
"string".start_with?("ing")          # => false
"string".start_with?("ing", "str")   # => true
