instance method String#hash

hash -> Integer[permalink][rdoc]

self のハッシュ値を返します。 eql? で等しい文字列は、常にハッシュ値も等しくなります。 

例

"test".hash                        # => 4038258770210371295
("te" + "st").hash == "test".hash  # => true
