instance method String#include?

include?(substr) -> bool[permalink][rdoc]

文字列中に部分文字列 substr が含まれていれば真を返します。

[PARAM] substr:
検索する文字列 
例

"hello".include? "lo"   #=> true
"hello".include? "ol"   #=> false
"hello".include? ?h     #=> true
