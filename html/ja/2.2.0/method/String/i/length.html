instance method String#length

length -> Integer[permalink][rdoc]
size -> Integer

文字列の文字数を返します。バイト数を知りたいときは bytesize メソッドを使ってください。 

例

"test".length   # => 4
"test".size     # => 4
"テスト".length   # => 3
"テスト".size     # => 3

[SEE_ALSO] String#bytesize
