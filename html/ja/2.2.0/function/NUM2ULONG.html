macro NUM2ULONG

unsigned long NUM2ULONG(VALUE x)

x を unsigned long 型の整数に変換します。

x が Fixnum、Float、Bignum オブジェクトのいずれでもな い場合は x.to_int による暗黙の型変換を試みます。

[EXCEPTION] TypeError:
x が nil の場合か、暗黙の型変換が成功しなかった場合に 発生します。
[EXCEPTION] RangeError:
x が unsigned long 型で表現できる値の範囲外であった場 合に発生します。
