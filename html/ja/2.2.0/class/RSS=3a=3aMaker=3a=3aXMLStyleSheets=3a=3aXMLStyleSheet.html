class RSS::Maker::XMLStyleSheets::XMLStyleSheet

クラスの継承リスト: RSS::Maker::XMLStyleSheets::XMLStyleSheet < Object < Kernel < BasicObject

要約

xml_stylesheetは maker.xml_stylesheet.new_xml_stylesheetで作成されたオ ブジェクトとします．

hrefとtypeは必ず指定する必要があります．ただし， hrefに設定した値の拡張子が.xslまたは.cssの場合は typeを指定する必要はありません．自動で推測します．
