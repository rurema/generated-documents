instance method Enumerable::Enumerator#rewind

rewind -> self[permalink][rdoc]

列挙状態を巻き戻します。

next メソッドによる外部列挙の状態を最初まで巻き戻します。 self を返します。

[SEE_ALSO] Enumerable::Enumerator#next

例: 

str = "xyz"
enum = str.each_byte

p enum.next # => 120
p enum.next # => 121
enum.rewind
p enum.next # => 120
