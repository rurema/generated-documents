instance method String#delete_prefix

delete_prefix(prefix) -> String[permalink][rdoc][edit]

文字列の先頭から prefix を削除した文字列のコピーを返します。

[PARAM] prefix:
先頭から削除する文字列を指定します。
[RETURN]
文字列の先頭から prefix を削除した文字列のコピー 

"hello".delete_prefix("hel") # => "lo"
"hello".delete_prefix("llo") # => "hello"
