instance method Range#to_s

to_s -> String[permalink][rdoc][edit]

self を文字列に変換します(始端と終端のオブジェクトは #to_s メソッドで文字列に変換されます)。

[SEE_ALSO] Range#inspect 

例

(1..5).to_s      # => "1..5"
("1".."5").to_s  # => "1..5"
