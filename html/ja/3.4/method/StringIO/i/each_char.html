instance method StringIO#each_char

each_char {|c| ... } -> self[permalink][rdoc][edit]
each_char -> Enumerator

自身に含まれる文字を一文字ずつブロックに渡して評価します。

自身は読み込み用にオープンされていなければなりません。

[EXCEPTION] IOError:
自身が読み込み用にオープンされていない場合に発生します。
