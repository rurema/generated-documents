singleton method Matrix.zero

zero(n) -> Matrix[permalink][rdoc][edit]

n × n の零行列（要素が全て 0 の行列）を生成して返します。 

例

require 'matrix'
p Matrix.zero(2) #=> Matrix[[0, 0], [0, 0]]
[PARAM] n:
生成する正方零行列の次数
zero(row, column) -> Matrix[permalink][rdoc][edit]

row × column の零行列（要素が全て 0 の行列）を生成して返します。 

例

require 'matrix'
p Matrix.zero(2, 3) #=> Matrix[[0, 0, 0], [0, 0, 0]]
[PARAM] row:
生成する行列の行数
[PARAM] column:
生成する行列の列数
