instance method Kernel#try_func

try_func(func, libs, headers = nil) -> bool[permalink][rdoc][edit] [added by mkmf]
try_func(func, libs, headers = nil) { ... } -> bool [added by mkmf]

関数 func がシステムに存在するかどうか検査します。 Kernel#have_func を使ってください。

[PARAM] func:
関数名を指定します。
[PARAM] libs:
ライブラリの名前を指定します。
[PARAM] headers:
関数 func を使用するのに必要なヘッダファイル名を指定します。これは関数の型をチェックするためではなく、関数が実際にはマクロで定義されている場合などのために使用します。
