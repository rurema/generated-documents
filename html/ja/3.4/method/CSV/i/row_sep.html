instance method CSV#row_sep

row_sep -> String[permalink][rdoc][edit]

行区切り文字列として使用する文字列を返します。 

例

require "csv"

csv = CSV.new("header1,header2|row1_1,row1_2", row_sep: "|")
csv.row_sep # => "|"
csv.read    # => [["header1", "header2"], ["row1_1", "row1_2"]]
