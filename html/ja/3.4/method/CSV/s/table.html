singleton method CSV.table

table(path, options = Hash.new) -> CSV::Table | [Array][permalink][rdoc][edit]

以下と同等のことを行うメソッドです。 


CSV.read( path, { headers:           true,
                  converters:        :numeric,
                  header_converters: :symbol }.merge(options) )
[PARAM] path:
ファイル名を指定します。
[PARAM] options:
CSV.new のオプションと同じオプションを指定できます。
