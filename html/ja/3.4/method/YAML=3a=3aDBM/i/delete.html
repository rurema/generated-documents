instance method YAML::DBM#delete

delete(key) -> object | nil[permalink][rdoc][edit]

key をキーとする要素を削除します。

削除した要素を返します。key に対応する値が見つからなかった場合は nil を返します。

[PARAM] key:
キーを文字列で指定します。
[EXCEPTION] DBMError:
要素の削除に失敗した場合に発生します。
