instance method Complex#to_f

to_f -> Float[permalink][rdoc][edit]

自身を Float に変換します。

[EXCEPTION] RangeError:
虚部が実数か、0 ではない場合に発生します。 
例

Complex(3).to_f    # => 3.0
Complex(3.5).to_f  # => 3.5
Complex(3, 2).to_f # => RangeError
