instance method Proc#binding

binding -> Binding[permalink][rdoc][edit]

Proc オブジェクトが保持するコンテキストを Binding オブジェクトで返します。 

例

def fred(param)
  proc {}
end

sample_proc = fred(99)
eval("param", sample_proc.binding)   # => 99
