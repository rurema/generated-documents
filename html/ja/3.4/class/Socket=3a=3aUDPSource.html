new(remote_addr, local_addr) {|msg| ... } -> Socket::UDPSource[permalink][rdoc][edit]

Socket::UDPSource オブジェクトを生成します。

このメソッドはユーザは直接使いません。Socket.udp_server_loop が内部で用います。
