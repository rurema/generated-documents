singleton method Rake::PackageTask.new

new(name = nil, version = nil) {|t| ... } -> Rake::PackageTask[permalink][rdoc]

自身を初期化してタスクを定義します。

ブロックが与えられた場合は、自身をブロックパラメータとしてブロックを評価します。

[PARAM] name:
パッケージ名を指定します。
[PARAM] version:
パッケージのバージョンを指定します。 ':noversion' というシンボルを指定するとバージョン情報をセットしません。
