singleton method IRB::MagicFile.open

open(path) -> File[permalink][rdoc]
open(path) { |io| ... } -> object

引数 path で指定したファイルを開いて、ファイル中のマジックコメントをエンコーディングに設定します。

ブロックを指定した場合はブロックの実行結果を返します。ブロックを指定しなかった場合はエンコーディングが設定された File オブジェクトを返します。

[PARAM] path:
パスを文字列で指定します。
