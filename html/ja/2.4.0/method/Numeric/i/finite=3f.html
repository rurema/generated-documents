instance method Numeric#finite?

finite? -> bool[permalink][rdoc]

self の絶対値が有限値の場合に true を、そうでない場合に false を返します。

例: 

10.finite?                      # => true
Rational(3).finite?             # => true

Float::INFINITY.finite?         # => false
Float::INFINITY.is_a?(Numeric)  # => true
