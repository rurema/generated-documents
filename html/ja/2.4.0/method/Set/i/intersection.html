instance method Set#&

intersection(enum) -> Set[permalink][rdoc]
self & enum -> Set

共通部分、すなわち、2つの集合のいずれにも属するすべての要素からなる新しい集合を作ります。

[PARAM] enum:
each メソッドが定義されたオブジェクトを指定します。
[EXCEPTION] ArgumentError:
引数 enum に each メソッドが定義されていない場合に発生します。 
require 'set'
s1 = Set[10, 20, 30]
s2 = Set[10, 30, 50]
p s1 & s2 #=> #<Set: {10, 30}>
