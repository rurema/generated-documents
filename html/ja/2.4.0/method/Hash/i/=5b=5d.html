instance method Hash#[]

self[key] -> object | nil[permalink][rdoc]

key に関連づけられた値を返します。

該当するキーが登録されていない時には、デフォルト値を返します。

デフォルト値と値としての nil を区別する必要がある場合は Hash#fetch または Hash#has_key? を使ってください。

[PARAM] key:
探索するキーを指定します。 
h = {:ab => "some" , :cd => "all"}
p h[:ab]             #=> "some"
p h[:ef]             #=> nil

h1 = Hash.new("default value")
p h1[:non]             #=> "default value"

h2 = Hash.new {|*arg| arg}
p h2[:non]             #=> [{}, :non]
