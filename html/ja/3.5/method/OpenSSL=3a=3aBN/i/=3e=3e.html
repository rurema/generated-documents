instance method OpenSSL::BN#>>

self >> other -> OpenSSL::BN[permalink][rdoc][edit]

自身を other ビット右シフトした値を返します。 


require 'openssl'

bn = 2.to_bn
bn >> 1    # => #<OpenSSL::BN 1>
bn         # => #<OpenSSL::BN 2>
[PARAM] other:
シフトするビット数
[EXCEPTION] OpenSSL::BNError:
計算時エラー
