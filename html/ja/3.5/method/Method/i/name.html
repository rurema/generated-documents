instance method Method#name

name -> Symbol[permalink][rdoc][edit]

このメソッドの名前を返します。 

例

class Foo
  def foo(arg)
    "foo called with arg #{arg}"
  end
end

m = Foo.new.method(:foo) # => #<Method: Foo#foo>
m.name # => :foo
