instance method Class#superclass

superclass -> Class | nil[permalink][rdoc][edit]

自身のスーパークラスを返します。 

例

File.superclass          #=> IO
IO.superclass            #=> Object
class Foo; end
class Bar < Foo; end
Bar.superclass           #=> Foo
Object.superclass        #=> BasicObject

ただし BasicObject.superclass は nil を返します。 

例

BasicObject.superclass   #=> nil
