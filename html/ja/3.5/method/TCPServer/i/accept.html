instance method TCPServer#accept

accept -> TCPSocket[permalink][rdoc][edit]

クライアントからの接続要求を受け付け、接続した TCPSocket のインスタンスを返します。

例: 

require 'socket'

TCPServer.open("", 0) {|serv|
  c = TCPSocket.new(*serv.addr.values_at(3,1))
  s = serv.accept
  c.write "foo"
  p s.recv(10)  #=> "foo"
}
