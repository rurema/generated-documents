instance method Object#psych_to_yaml

to_yaml(options = {}) -> String[permalink][rdoc][edit] [added by psych]
psych_to_yaml(options = {}) -> String [added by psych]

オブジェクトを YAML document に変換します。

options でオプションを指定できます。 Psych.dump と同じなので詳しくはそちらを参照してください。

syck に to_yaml メソッドがあるため、 psych_to_yaml が別名として定義されています。将来的に syck が廃止された場合 psych_to_yaml は廃止される予定であるため、特別の事情がない限り to_yaml を用いてください。

[PARAM] options:
出力オプション
