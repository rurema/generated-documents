instance method JSON::State#array_nl=

array_nl=(str)[permalink][rdoc][edit]

JSON の配列の後に出力する文字列をセットします。 

例

require "json"

json_state = JSON::State.new({})
json_state.array_nl        # => ""
json_state.array_nl = "\n"
json_state.array_nl        # => "\n"
