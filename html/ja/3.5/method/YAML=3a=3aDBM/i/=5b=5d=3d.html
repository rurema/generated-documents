instance method YAML::DBM#[]=

self[key] = value[permalink][rdoc][edit]
store(key, value) -> object

key に対して value を格納します。

val で指定したオブジェクトを返します。

[PARAM] key:
キーを文字列で指定します。
[PARAM] value:
値を指定します。YAML 形式に変換できるオブジェクトが指定できます。
[EXCEPTION] DBMError:
要素の格納に失敗した場合に発生します。
