instance method CGI::HtmlExtension#submit

submit(value = nil, name = nil) -> String[permalink][rdoc][edit]

タイプが submit である input 要素を生成します。

[PARAM] value:
value 属性の値を指定します。
[PARAM] name:
name 属性の値を指定します。

例: 

submit
  # <INPUT TYPE="submit">

submit("ok")
  # <INPUT TYPE="submit" VALUE="ok">

submit("ok", "button1")
  # <INPUT TYPE="submit" VALUE="ok" NAME="button1">
submit(attributes) -> String[permalink][rdoc][edit]

タイプが submit である input 要素を生成します。

[PARAM] attributes:
属性をハッシュで指定します。

例: 

submit({ "VALUE" => "ok", "NAME" => "button1", "ID" => "foo" })
  # <INPUT TYPE="submit" VALUE="ok" NAME="button1" ID="foo">
