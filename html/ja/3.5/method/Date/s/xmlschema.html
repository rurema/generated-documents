singleton method Date.xmlschema

xmlschema(str = '-4712-01-01', start = Date::ITALY) -> Date[permalink][rdoc][edit]

XML Schema による書式の日付を解析し、その情報に基づいて日付オブジェクトを生成します。

Date._xmlschema も参照してください。

[PARAM] str:
日付をあらわす文字列
[PARAM] start:
グレゴリオ暦をつかい始めた日をあらわすユリウス日
