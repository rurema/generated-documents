class REXML::Declaration

クラスの継承リスト: REXML::Declaration < REXML::Child < REXML::Node < Object < Kernel < BasicObject

要約

DTD に含まれる各種宣言ノードを表すクラスです。

このクラス自体は直接はインスタンスを作りません。 各サブクラスのインスタンスが使われます。

目次

インスタンスメソッド
to_s write

インスタンスメソッド

to_s -> String[permalink][rdoc]

ノードを文字列化します。

write(output, indent) -> ()[permalink][rdoc]

output にノードを出力します。

このメソッドは deprecated です。REXML::Formatter で 出力してください。

[PARAM] output:
出力先の IO オブジェクト
[PARAM] indent:
インデントの大きさ。無視されます。
