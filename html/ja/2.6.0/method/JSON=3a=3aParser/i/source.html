instance method JSON::Parser#source

source -> String[permalink][rdoc][edit]

現在のソースのコピーを返します。 

例

require 'json'

parser = JSON::Parser.new(DATA.read)
print parser.source

# =>     "name":"tanaka",
# =>     "age":20

# =>   "Suzuki": {
# =>     "name":"suzuki",

# => }
