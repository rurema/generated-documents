instance method Symbol#id2name

id2name -> String[permalink][rdoc][edit]
to_s -> String

シンボルに対応する文字列を返します。

逆に、文字列に対応するシンボルを得るには String#intern を使います。 

p :foo.id2name  # => "foo"
p :foo.id2name.intern == :foo  # => true
