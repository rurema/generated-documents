instance method Object#<=>

self <=> other -> 0 | nil[permalink][rdoc][edit]

self === other である場合に 0 を返します。そうでない場合には nil を返します。 

例

a = Object.new
b = Object.new
a <=> a # => 0
a <=> b # => nil
