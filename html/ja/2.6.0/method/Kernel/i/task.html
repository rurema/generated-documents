instance method Kernel#task

task(*args) { ... } -> Rake::Task[permalink][rdoc][edit] [added by rake]

Rake タスクを定義します。

[PARAM] args:
タスク名と依存タスクを指定します。

例: 

task :clobber => [:clean] do
  rm_rf "html"
end
