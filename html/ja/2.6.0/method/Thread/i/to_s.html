instance method Thread#inspect

inspect -> String[permalink][rdoc][edit]
to_s -> String

自身を人間が読める形式に変換した文字列を返します。 

例

a = Thread.current
a.inspect   # => "#<Thread:0x00007fdbaf07ddb0 run>"
b = Thread.new{}
b.inspect   # => "#<Thread:0x00007fdbaf8f7d10@(irb):3 dead>"
