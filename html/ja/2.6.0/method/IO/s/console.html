singleton method IO.console

console -> File | nil[permalink][rdoc][edit] [added by io/console]

端末を File オブジェクトで返します。 

require "io/console"
IO.console # => #<File:/dev/tty>

プロセスが端末から切り離された状態で実行すると nil を返します。

戻り値はプラットフォームや環境に依存します。
