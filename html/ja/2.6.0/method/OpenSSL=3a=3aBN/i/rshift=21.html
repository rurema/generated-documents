instance method OpenSSL::BN#rshift!

rshift!(n) -> self[permalink][rdoc][edit]

自身を n ビット右シフトします。 [[m:OpenSSL::BN#>>]と異なり、破壊的メソッドです。 


require 'openssl'

bn = 8.to_bn
bn.rshift!(1)    # => #<OpenSSL::BN 4>
bn               # => #<OpenSSL::BN 4>
[PARAM] n:
シフトするビット数
[EXCEPTION] OpenSSL::BNError:
計算時エラー
