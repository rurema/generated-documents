instance method Hash#length

length -> Integer[permalink][rdoc][edit]
size -> Integer

ハッシュの要素の数を返します。 


h = { "d" => 100, "a" => 200, "v" => 300, "e" => 400 }
h.length        #=> 4
h.size          #=> 4
h.delete("a")   #=> 200
h.length        #=> 3
h.size          #=> 3
