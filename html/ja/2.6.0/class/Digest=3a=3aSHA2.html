特異メソッド

new(bitlen = 256) -> Digest::SHA2[permalink][rdoc][edit]

与えられた bitlen に対応する SHA2 ハッシュを生成するためのオブジェクトを内部で設定して自身を初期化します。

[PARAM] bitlen:
ハッシュの長さを指定します。256, 384, 512 が指定可能です。
[EXCEPTION] ArgumentError:
bitlen に 256, 384, 512 以外の値を指定した場合に発生します。

インスタンスメソッド

block_length -> Integer[permalink][rdoc][edit]

ダイジェストのブロック長を返します。

digest_length -> Integer[permalink][rdoc][edit]

ダイジェストのハッシュ値のバイト長を返します。
