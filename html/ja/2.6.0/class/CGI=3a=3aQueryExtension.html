インスタンスメソッド

self[key] -> Array[permalink][rdoc][edit]

文字列 key に対応するパラメータを配列で返します。 key に対応するパラメータが見つからなかった場合は、nil を返します。（CGI#paramsと等価です）

フォームから入力された値や、URL に埋め込まれた QUERY_STRING のパース結果の取得などに使用します。

[PARAM] key:
キーを文字列で指定します。
accept -> String[permalink][rdoc][edit]

ENV['HTTP_ACCEPT'] を返します。

content_length -> Integer[permalink][rdoc][edit]

ENV['CONTENT_TYPE'] を返します。

cookies -> Hash[permalink][rdoc][edit]

クッキーの名前と値をペアにした要素を持つハッシュを返します。

[PARAM] value:

from -> String[permalink][rdoc][edit]

ENV['HTTP_FROM'] を返します。

gateway_interface -> String[permalink][rdoc][edit]

ENV['GATEWAY_INTERFACE'] を返します。

has_key?(*args) -> bool[permalink][rdoc][edit]
key?(*args) -> bool
include?(*args) -> bool

与えられたキーがクエリに含まれている場合は、真を返します。そうでない場合は、偽を返します。

[PARAM] args:
キーを一つ以上指定します。
host -> String[permalink][rdoc][edit]

ENV['HTTP_HOST'] を返します。

keys(*args) -> [String][permalink][rdoc][edit]

すべてのパラメータのキーを配列として返します。

multipart? -> bool[permalink][rdoc][edit]

マルチパートフォームの場合は、真を返します。そうでない場合は、偽を返します。 

例：
cgi = CGI.new
if cgi.multipart?
  field1=cgi['field1'].read
else
  field1=cgi['field1']
end
negotiate -> String[permalink][rdoc][edit]

ENV['HTTP_NEGOTIATE'] を返します。

params -> Hash[permalink][rdoc][edit]

パラメータを格納したハッシュを返します。

フォームから入力された値や、URLに埋め込まれた QUERY_STRING のパース結果の取得などに使用します。 

cgi = CGI.new
cgi.params['developer']     # => ["Matz"] (Array)
cgi.params['developer'][0]  # => "Matz"
cgi.params['']              # => nil
