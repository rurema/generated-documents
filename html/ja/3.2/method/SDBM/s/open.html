singleton method SDBM.open

open(dbname, mode = 0666) -> SDBM[permalink][rdoc][edit]
open(dbname, mode = 0666) {|db| ... } -> object

dbname で指定したデータベースをモードを mode に設定してオープンします。

[PARAM] dbname:
データベースの名前を指定します。
[PARAM] mode:
省略値は 0666 です。mode として nil を指定するとデータベースが存在しない時には新たなデータベースを作らず nil を返します。
