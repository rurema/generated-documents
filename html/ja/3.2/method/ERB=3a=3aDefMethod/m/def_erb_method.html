module function ERB::DefMethod.#def_erb_method

def_erb_method(methodname, erb) -> nil[permalink][rdoc][edit]

self に erb のスクリプトをメソッドとして定義します。

メソッド名は methodname で指定します。 erb が文字列の時、そのファイルを読み込み ERB で変換したのち、メソッドとして定義します。

[PARAM] methodname:
メソッド名
[PARAM] erb:
ERBインスタンスもしくはERBソースファイル名

例: 

require 'erb'
class Writer
  extend ERB::DefMethod
  def_erb_method('to_html', 'writer.erb')
  ...
end
...
puts writer.to_html
