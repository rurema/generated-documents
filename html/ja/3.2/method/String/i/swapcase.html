instance method String#swapcase

swapcase(*options) -> String[permalink][rdoc][edit]

大文字を小文字に、小文字を大文字に変更した文字列を返します。

[PARAM] options:
オプションの詳細は String#downcase を参照してください。 
例

p "ABCxyz".swapcase   # => "abcXYZ"
p "Access".swapcase   # => "aCCESS"
