instance method Hash#except

except(*keys) -> Hash[permalink][rdoc][edit]

引数で指定された以外のキーとその値だけを含む Hash を返します。

引数に指定されていて Hash に存在しないキーは無視されます。 


h = { a: 100, b: 200, c: 300 }
h.except(:a) # => {:b=>200, :c=>300}
