instance method Hash#>

self > other -> bool[permalink][rdoc][edit]

other が self のサブセットである場合に真を返します。

[PARAM] other:
自身と比較したい Hash オブジェクトを指定します。 
例

h1 = {a:1, b:2}
h2 = {a:1, b:2, c:3}
h1 > h2    # => false
h2 > h1    # => true
h1 > h1    # => false
