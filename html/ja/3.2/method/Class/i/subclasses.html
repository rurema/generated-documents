instance method Class#subclasses

subclasses -> [Class][permalink][rdoc][edit]

自身が直接のスーパークラスになっている(特異クラスを除く)クラスの配列を返します。返り値の配列の順序は未定義です。 

例

class A; end
class B < A; end
class C < B; end
class D < A; end

A.subclasses        # => [D, B]
B.subclasses        # => [C]
C.subclasses        # => []
