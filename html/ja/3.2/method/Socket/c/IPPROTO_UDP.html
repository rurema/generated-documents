constant Socket::IPPROTO_UDP

IPPROTO_UDP -> Integer[permalink][rdoc][edit]

User Datagram Protocol。 BasicSocket#getsockopt, BasicSocket#setsockopt の level 引数に使用します。

また、Socket.open の protocol 引数に渡す利用法もあります。
