instance method CSV::Table#mode

mode -> Symbol[permalink][rdoc][edit]

現在のアクセスモードを返します。 

例

require "csv"

row = CSV::Row.new(["header1", "header2"], ["row1_1", "row1_2"])
table = CSV::Table.new([row])
table.mode    # => :col_or_row
table.by_col!
table.mode    # => :col
