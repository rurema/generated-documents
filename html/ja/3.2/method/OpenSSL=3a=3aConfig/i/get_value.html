instance method OpenSSL::Config#get_value

get_value(section, name) -> String | nil[permalink][rdoc][edit]

オブジェクトが持っている設定情報を返します。

キーに対応する設定情報がない場合は nil を返します。

[PARAM] section:
セクションを表す文字列。"" を渡すことでグローバルな設定情報を読むことができます。
[PARAM] name:
キーを表す文字列
