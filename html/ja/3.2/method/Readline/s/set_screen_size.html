singleton method Readline.set_screen_size

set_screen_size(rows, columns) -> Readline[permalink][rdoc][edit]

端末のサイズを引数 row、columns に設定します。

[PARAM] rows:
行数を整数で指定します。
[PARAM] columns:
列数を整数で指定します。
[EXCEPTION] NotImplementedError:
サポートしていない環境で発生します。

[SEE_ALSO] GNU Readline ライブラリの rl_set_screen_size 関数
