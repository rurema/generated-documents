instance method UnboundMethod#hash

hash -> Integer[permalink][rdoc][edit]

自身のハッシュ値を返します。 

例

a = method(:==).unbind
b = method(:eql?).unbind
p a.eql? b          # => true
p a.hash == b.hash  # => true
p [a, b].uniq.size  # => 1
