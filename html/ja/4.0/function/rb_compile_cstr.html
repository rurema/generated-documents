function rb_compile_cstr

[edit]
NODE * rb_compile_cstr(const char *f, const char *s, int len, int line)

C の文字列 s を構文木にコンパイルし、ruby_eval_tree と ruby_eval_tree_begin に格納します。ruby_eval_tree を返します。またコンパイルするときにファイル f の line 行目からをコンパイルしていると仮定します。
