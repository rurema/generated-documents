class RubyVM

クラスの継承リスト: RubyVM < Object < Kernel < BasicObject

要約

Ruby の 内部情報へのアクセス手段を提供するクラスです。 デバッグ用、プロトタイピング用、研究用などの とても限定された用途向けです。 一般ユーザーは使うべきではありません。

目次

定数
DEFAULT_PARAMS INSTRUCTION_NAMES OPTS

定数

DEFAULT_PARAMS -> {Symbol => Integer}[permalink][rdoc]

RubyVM のデフォルトのパラメータを返します。

[注意] この値は C Ruby 固有のものです。変更しても RubyVM の動作に は影響しません。また、仕様は変更される場合があるため、この値に依存すべ きではありません。

INSTRUCTION_NAMES -> [String][permalink][rdoc]

RubyVM の命令シーケンスの名前の一覧を返します。

[SEE_ALSO] RubyVM::InstructionSequence

OPTS -> [String][permalink][rdoc]

RubyVM のビルドオプションの一覧を返します。
