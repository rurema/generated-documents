class OpenSSL::ASN1::Boolean

クラスの継承リスト: OpenSSL::ASN1::Boolean < OpenSSL::ASN1::Primitive < OpenSSL::ASN1::ASN1Data < Object < Kernel < BasicObject

要約

ASN.1 の Boolean 型(Universal タグのタグ番号1)を表すクラスです。

特異メソッド

new(value) -> OpenSSL::ASN1::Boolean[permalink][rdoc]
new(value, tag, tagging, tag_class) -> OpenSSL::ASN1::Boolean

ASN.1 の Boolean 型の値を表現する OpenSSL::ASN1::Boolean オブジェクトを 生成します。

value 以外の引数を省略した場合はタグクラスは :UNIVERSAL、 タグ は OpenSSL::ASN1::BOOLEAN となります。

[PARAM] value:
ASN.1 値を表す Ruby のオブジェクト(true もしくは false)
[PARAM] tag:
タグ番号
[PARAM] tagging:
タグ付けの方法(:IMPLICIT もしくは :EXPLICIT)
[PARAM] tag_class:
タグクラス(:UNIVERSAL, :CONTEXT_SPECIFIC, :APPLICATION, :PRIVATE のいずれか)
