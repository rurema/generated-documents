class Net::HTTPServerException

クラスの継承リスト: Net::HTTPServerException < Net::HTTPExceptions < Net::ProtoServerError < Net::ProtocolError < StandardError < Exception < Object < Kernel < BasicObject

要約

HTTP ステータスコード 4xx を受け取ったという例外です。

クライアントのリクエストに誤りがあるか、サーバにリクエストを拒否さ れた(認証が必要、リソースが存在しないなどで)ことを示します。
