module function Math.#cbrt

cbrt(x) -> Float[permalink][rdoc]

x の立方根を返します。

[PARAM] x:
実数
[EXCEPTION] TypeError:
xに数値以外を指定した場合に発生します。
[EXCEPTION] RangeError:
xに実数以外の数値を指定した場合に発生します。 
例

-9.upto(9) {|x|
  p [x, Math.cbrt(x), Math.cbrt(x)**3]
}
