instance method UnboundMethod#inspect

inspect -> String[permalink][rdoc]
to_s -> String

self を読みやすい文字列として返します。

詳しくは Method#inspect を参照してください。 

例

String.instance_method(:count).inspect   # => "#<UnboundMethod: String#count>"
