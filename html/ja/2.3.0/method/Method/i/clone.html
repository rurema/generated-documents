instance method Method#clone

clone -> Method[permalink][rdoc]

自身を複製した Method オブジェクトを作成して返します。

例: 

class Foo
  def foo
    "foo"
  end
end

m = Foo.new.method(:foo) # => #<Method: Foo#foo>
m.call # => "foo"

m.clone # => #<Method: Foo#foo>
m.clone.call # => "foo"
