instance method Gem::RemoteFetcher#connection_for

connection_for(uri) -> Net::HTTP[permalink][rdoc]

HTTP コネクションを生成して返します。

既に接続している URI であれば、生成済みのコネクションを返します。 また、必要があればプロキシを使用します。

[PARAM] uri:
接続先の URI を指定します。
