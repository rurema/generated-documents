instance method IO#sync

sync -> bool[permalink][rdoc]

現在の出力が同期モードならば true を返します。そうでない場合は false を返します。

[EXCEPTION] IOError:
既に close されていた場合に発生します。 
例

File.open("testfile", "w") do |f|
  f.sync      # => false
  f.sync = true
  f.sync      # => true
end
