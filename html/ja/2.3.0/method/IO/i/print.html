instance method IO#print

print(*arg) -> nil[permalink][rdoc]

引数を IO ポートに順に出力します。引数を省略した場合は、$_ を出力します。

[PARAM] arg:
Kernel.#print と同じです。
[EXCEPTION] IOError:
自身が書き込み用にオープンされていなければ発生します。
[EXCEPTION] Errno::EXXX:
出力に失敗した場合に発生します。 
例

$stdout.print("This is ", 100, " percent.\n") # => This is 100 percent.
