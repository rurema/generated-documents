instance method Addrinfo#ip_address

ip_address -> String[permalink][rdoc]

IP アドレスを文字列で返します。 

require 'socket'

Addrinfo.tcp("127.0.0.1", 80).ip_address    #=> "127.0.0.1"
Addrinfo.tcp("::1", 80).ip_address          #=> "::1"
