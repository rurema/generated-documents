instance method Kernel#jj

jj(*objects) -> nil[permalink][rdoc] [added by json]

与えられたオブジェクトを JSON 形式の文字列で標準出力に人間に読みやすく整形して出力します。

[PARAM] objects:
JSON 形式で出力したいオブジェクトを指定します。 
例

require "json"

jj([1,2,{"name" => "tanaka","age" => 19}])
# => [
#      1,
#      2,
#      {
#        "name": "tanaka",
#        "age": 19
#      }
#    ]
