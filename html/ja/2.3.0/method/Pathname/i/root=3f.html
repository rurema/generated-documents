instance method Pathname#root?

root? -> bool[permalink][rdoc]

self がルートディレクトリであれば真を返します。判断は文字列操作によっ て行われ、ファイルシステムはアクセスされません。 

例

require 'pathname'

Pathname('/').root?         # => true
Pathname('/im/sure').root?  # => false
