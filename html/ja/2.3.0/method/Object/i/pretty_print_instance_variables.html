instance method Object#pretty_print_instance_variables

pretty_print_instance_variables -> [String | Symbol][permalink][rdoc] [added by pp]

プリティプリント時に表示すべき自身のインスタンス変数名の配列をソートして返します。 返されたインスタンス変数はプリティプリント時に表示されます。

pp に表示したくないインスタンス変数がある場合にこのメソッドを再定義します。
