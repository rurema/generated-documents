instance method Logger#level

level -> Integer[permalink][rdoc]
sev_threshold -> Integer

レシーバにセットされているログレベルを取得します。 

例

require 'logger'

logger = Logger.new(STDOUT)
logger.level # => 0
logger.level = Logger::Severity::ERROR
logger.level # => 3
