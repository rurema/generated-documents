instance method Numeric#imag

imag -> 0[permalink][rdoc]
imaginary -> 0

常に 0 を返します。

例: 

12.imag     # => 0
-12.imag    # => 0
1.2.imag    # => 0
-1.2.imag   # => 0

Numeric のサブクラスは、このメソッドを適切に再定義しなければなりません。

[SEE_ALSO] Numeric#real、Complex#imag
