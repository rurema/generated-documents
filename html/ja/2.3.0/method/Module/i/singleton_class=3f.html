instance method Module#singleton_class?

singleton_class? -> bool[permalink][rdoc]

self が特異クラスの場合に true を返します。そうでなければ false を返し ます。 

class C
end
C.singleton_class?                  # => false
C.singleton_class.singleton_class?  # => true
