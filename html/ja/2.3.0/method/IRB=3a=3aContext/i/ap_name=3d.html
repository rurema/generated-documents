instance method IRB::Context#ap_name=

ap_name=(val)[permalink][rdoc]

自身のアプリケーション名を val に設定します。

.irbrc ファイル中で IRB.conf[:AP_NAME] を設定する事でも同様の操作が行え ます。

[PARAM] val:
アプリケーション名を String で指定します。
