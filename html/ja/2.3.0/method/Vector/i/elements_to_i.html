instance method Vector#elements_to_i

elements_to_i -> Vector[permalink][rdoc]

ベクトルの各成分をIntegerに変換したベクトルを返します。

このメソッドは deprecated です。 map(&:to_i) を使ってください。 

例

require 'matrix'
v = Vector.elements([2.5, 3.0, 5.01, 7])
vi = v.elements_to_i
p vi # => "Vector[2, 3, 5, 7]"
