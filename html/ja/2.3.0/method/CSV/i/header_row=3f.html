instance method CSV#header_row?

header_row? -> bool[permalink][rdoc]

次に読み込まれる行が、ヘッダである場合に真を返します。 そうでない場合は、偽を返します。 

例

require "csv"

csv = CSV.new("header1,header2\nrow1_1,row1_2", headers: true)
csv.header_row? # => true
csv.readline
csv.header_row? # => false
