instance method CSV::Row#to_csv

to_csv -> String[permalink][rdoc]
to_s -> String

自身を CSV な文字列として返します。ヘッダは使用しません。 

例

require "csv"

row = CSV::Row.new(["header1", "header2"], [1, 2])
row.to_csv                                    # => "1,2\n"
row.to_csv( {col_sep: "|", row_sep: "<br>"} ) # => "1|2<br>"
