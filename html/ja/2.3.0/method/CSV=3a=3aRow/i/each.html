instance method CSV::Row#each

each {|header, field| ... } -> self[permalink][rdoc]

与えられたブロックにヘッダとフィールドの組を渡して評価します。

[RETURN]
メソッドチェーンのために自身を返します。 
例

require "csv"

row = CSV::Row.new(["header1", "header2", "header3", "header4"], [1, 2, 3, 4])
row.each { |header, field| puts "#{header} - #{field}" }

# => header1 - 1
# => header2 - 2
# => header3 - 3
# => header4 - 4
