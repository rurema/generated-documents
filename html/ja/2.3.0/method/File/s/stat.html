singleton method File.stat

stat(filename) -> File::Stat[permalink][rdoc]

filename の情報を含む File::Stat オブジェクトを生成し て返します。

[PARAM] filename:
ファイル名を表す文字列を指定します。
[EXCEPTION] Errno::EXXX:
情報の取得に失敗した場合に発生します。

例: 

File.stat("testfile").class   # => File::Stat
File.stat("testfile").mtime   # => 2017-12-10 01:13:56 +0900

[SEE_ALSO] IO#stat, File#lstat
