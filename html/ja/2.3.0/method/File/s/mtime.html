singleton method File.mtime

mtime(filename) -> Time[permalink][rdoc]

最終更新時刻を返します。

[PARAM] filename:
ファイル名を表す文字列か IO オブジェクトを指定します。
[EXCEPTION] Errno::EXXX:
ファイルの時刻の取得に失敗した場合に発生します。

例: 

File.mtime(__FILE__) # => 2017-12-03 03:16:22 +0900
