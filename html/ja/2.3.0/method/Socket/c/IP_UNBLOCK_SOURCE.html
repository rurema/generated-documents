constant Socket::IP_UNBLOCK_SOURCE

IP_UNBLOCK_SOURCE -> Integer[permalink][rdoc]

Unblock IPv4 multicast packets with a give source address。 BasicSocket#getsockopt, BasicSocket#setsockopt の第2引数(optname)に使用します。
