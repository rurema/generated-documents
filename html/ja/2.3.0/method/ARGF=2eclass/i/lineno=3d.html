instance method ARGF.class#lineno=

lineno=(number)[permalink][rdoc]

全引数ファイルを一つのファイルとみなしたときの現在の行番号を number に書き換えます。

この値を読み込みたい場合は ARGF.class#lineno を使用してください。

[PARAM] number:
更新後の行番号を指定する 
ARGF.lineno      # => 0
ARGF.readline    # => "This is line 1\n"
ARGF.lineno      # => 1
ARGF.lineno = 0  # => 0
ARGF.lineno      # => 0
