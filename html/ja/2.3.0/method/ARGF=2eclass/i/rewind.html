instance method ARGF.class#rewind

rewind -> 0[permalink][rdoc]

ARGFが現在開いているファイルのファイルポインタを先頭に戻します。 

ARGF.readline   # => "This is line one\n"
ARGF.rewind     # => 0
ARGF.lineno     # => 0
ARGF.readline   # => "This is line one\n"
