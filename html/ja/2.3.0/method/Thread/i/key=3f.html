instance method Thread#key?

key?(name) -> bool[permalink][rdoc]

name に対応したスレッドに固有のデータが定義されていれば true を返します。

[PARAM] name:
文字列か Symbol で指定します。 
例

me = Thread.current
me[:oliver] = "a"
me.key?(:oliver)    # => true
me.key?(:stanley)   # => false
