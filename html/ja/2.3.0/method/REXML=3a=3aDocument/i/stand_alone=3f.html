instance method REXML::Document#stand_alone?

stand_alone? -> String[permalink][rdoc]

XML 宣言の standalone の値を文字列で返します。

例

require 'rexml/document'
doc = REXML::Document.new(<<EOS)
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<e />
EOS
doc.stand_alone? # => "yes"
