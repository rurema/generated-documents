library webrick/httpproxy

要約

プロクシの機能を提供するライブラリです。CONNECT メソッドにも対応しています。

クラス

WEBrick::HTTPProxyServer

プロクシの機能を提供するクラスです。CONNECT メソッドにも対応しています。

オブジェクト

WEBrick::NullReader

空のソケットのふりをするオブジェクトです。

webrick/config
webrick/cookie
webrick/htmlutils

webrick/httprequest

webrick/httpresponse

HTTP サーバの機能を提供するライブラリです。

webrick/httpservlet

webrick/httpservlet/abstract
