library bigdecimal/ludcmp

要約

LU 分解を用いて、連立1次方程式 Ax = b の解 x を求める機能を提供します。

Ruby のソースコード中の以下のサンプルスクリプトも参考にしてください。

  • https://github.com/ruby/ruby/blob/trunk/ext/bigdecimal/sample/linear.rb

モジュール

LUSolve

LU 分解を用いて、連立1次方程式 Ax = b の解 x を求めるモジュールです。

同時にrequireされるライブラリ

bigdecimal

bigdecimal は浮動小数点数演算ライブラリです。 任意の精度で 10 進表現された浮動小数点数を扱えます。
