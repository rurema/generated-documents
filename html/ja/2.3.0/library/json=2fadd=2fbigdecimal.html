library json/add/bigdecimal

要約

BigDecimal に JSON 形式の文字列に変換するメソッドや JSON 形式の文字列から Ruby のオブジェクトに変換するメソッドを定義します。

同時にrequireされるライブラリ

bigdecimal

bigdecimal は浮動小数点数演算ライブラリです。 任意の精度で 10 進表現された浮動小数点数を扱えます。

追加・再定義されるメソッド

BigDecimal#to_json BigDecimal.json_create
