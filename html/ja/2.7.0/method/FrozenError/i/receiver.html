instance method FrozenError#receiver

receiver -> object[permalink][rdoc][edit]

self が発生した時のレシーバオブジェクトを返します。

[EXCEPTION] ArgumentError:
レシーバが設定されていない時に発生します。 

begin
  [1, 2, 3].freeze << 4
rescue FrozenError => err
  p err.receiver  # => [1, 2, 3]
end
