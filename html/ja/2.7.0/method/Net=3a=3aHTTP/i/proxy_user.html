instance method Net::HTTP#proxy_user

proxy_user -> String|nil[permalink][rdoc][edit]

プロクシ経由で接続し、さらにプロクシのユーザ認証をする HTTP オブジェクトなら認証のユーザ名をを返します。

そうでないなら nil を返します。
