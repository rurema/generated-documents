module function Math.#erfc

erfc(x) -> Float[permalink][rdoc][edit]

x の相補誤差関数（complementary error function）の値を返します。

[PARAM] x:
実数
[EXCEPTION] TypeError:
x に数値以外を指定した場合に発生します。
[EXCEPTION] RangeError:
x に実数以外の数値を指定した場合に発生します。 
例

Math.erfc(0) # => 1.0
