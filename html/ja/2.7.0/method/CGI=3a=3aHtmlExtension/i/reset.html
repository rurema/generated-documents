instance method CGI::HtmlExtension#reset

reset(value = nil, name = nil) -> String[permalink][rdoc][edit]

タイプが reset である input 要素を生成します。

[PARAM] value:
value 属性の値を指定します。
[PARAM] name:
name 属性の値を指定します。

例: 

reset
  # <INPUT TYPE="reset">

reset("reset")
  # <INPUT TYPE="reset" VALUE="reset">
reset(attributes) -> String[permalink][rdoc][edit]

タイプが reset である input 要素を生成します。

[PARAM] attributes:
属性をハッシュで指定します。 
reset({ "VALUE" => "reset", "ID" => "foo" })
  # <INPUT TYPE="reset" VALUE="reset" ID="foo">
