instance method Array#<<

self << obj -> self[permalink][rdoc][edit]

指定された obj を自身の末尾に破壊的に追加します。 

例

ary = [1]
ary << 2
p ary      # [1, 2]

またこのメソッドは self を返すので、以下のように連続して書くことができます。 

例

ary = [1]
ary << 2 << 3 << 4
p ary   #=> [1, 2, 3, 4]
[PARAM] obj:
自身に加えたいオブジェクトを指定します。Array#push と違って引数は一つしか指定できません。
