instance method Kernel#xpopen

xpopen(command, *mode) -> IO[permalink][rdoc][edit] [added by mkmf]
xpopen(command, *mode) { ... } -> object [added by mkmf]

command を表示してから IO.popen の実行します。

[PARAM] command:
コマンド名を指定します。
[PARAM] mode:
オープンする IO ポートのモードを指定します。mode の詳細は Kernel.#open 参照して下さい。
