instance method Array#intersect?

intersect?(other) -> bool[permalink][rdoc][edit]

other と共通の要素が少なくとも1個あれば true を、なければ false を返します。 

例

a = [ 1, 2, 3 ]
b = [ 3, 4, 5 ]
c = [ 5, 6, 7 ]
a.intersect?(b)   # => true
a.intersect?(c)   # => false
