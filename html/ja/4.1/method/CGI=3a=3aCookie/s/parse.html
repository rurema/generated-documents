singleton method CGI::Cookie.parse

parse(raw_cookie) -> Hash[permalink][rdoc][edit]

クッキー文字列をパースします。

[PARAM] raw_cookie:
生のクッキーを表す文字列を指定します。 
例：
cookies = CGI::Cookie.parse("raw_cookie_string")
  # { "name1" => cookie1, "name2" => cookie2, ... }
