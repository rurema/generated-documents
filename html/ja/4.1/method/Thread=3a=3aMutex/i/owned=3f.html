instance method Thread::Mutex#owned?

owned? -> bool[permalink][rdoc][edit]

self がカレントスレッドによってロックされている場合に true を返します。そうでない場合に false を返します。 

例

m = Mutex.new
m.owned? # => false
m.lock
Thread.new do
  m.owned? # => false
end.join
m.owned? # => true
