constant Socket::IP_PORTRANGE

IP_PORTRANGE -> Integer[permalink][rdoc][edit]

Set the port range for sockets with unspecified port numbers。 BasicSocket#getsockopt, BasicSocket#setsockopt の第2引数(optname)に使用します。
