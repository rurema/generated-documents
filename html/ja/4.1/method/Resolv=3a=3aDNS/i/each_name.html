instance method Resolv::DNS#each_name

each_name(address) {|name| ...} -> ()[permalink][rdoc][edit]

IP アドレス address のホスト名をルックアップし、各ルックアップ結果のホスト名に対してブロックを評価します。

[PARAM] address:
IPアドレスを文字列、 Resolv::IPv4 のインスタンス、 Resolv::IPv6 のインスタンス、のいずれか与えます。
