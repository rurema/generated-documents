instance method Vector#each

each {|e| ... } -> self[permalink][rdoc][edit]
each -> Enumerator

ベクトルの各要素をブロックの引数として順にブロックを呼び出します。

このクラスは Enumerable を include しているため、このメソッドを経由して Enumerable の各メソッドを使うことができます。

ブロックを省略した場合は Enumerator を返します。
