instance method REXML::Namespace#expanded_name

expanded_name -> String[permalink][rdoc][edit]

REXML::Namespace#name= で設定された名前を返します。

name= で指定した名前が prefix を含んでいれば prefix を含む名前を返し、そうでなければ prefix を含まない名前を返します。
