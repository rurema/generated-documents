instance method Method#==

self == other -> bool[permalink][rdoc][edit]
eql?(other) -> bool

自身と other が同じインスタンスの同じメソッドを表す場合に true を返します。そうでない場合に false を返します。

[PARAM] other:
自身と比較したいオブジェクトを指定します。 
例

s = "bar"
a = s.method(:size)
b = s.method(:size)
p a == b                            #=> true
