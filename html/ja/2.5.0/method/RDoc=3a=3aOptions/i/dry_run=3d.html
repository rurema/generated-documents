instance method RDoc::Options#dry_run=

dry_run=(val)[permalink][rdoc][edit]

val に true を指定した場合、コマンドライン引数の --dry-run オプションと同様の指定を行います。

[PARAM] val:
--dry-run オプションと同じ指定を行う場合は true、そうでない場合は false を指定します。
