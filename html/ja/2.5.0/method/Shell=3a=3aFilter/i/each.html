instance method Shell::Filter#each

each(rs = nil) -> ()[permalink][rdoc][edit]

フィルタの一行ずつをblockに渡します。

[PARAM] rs:
レコードセパレーターを表す文字列を指定します。 nil ならば、Shell.record_separatorの値が使用されます。

使用例 

require 'shell'
sh = Shell.new
sh.cat("/etc/passwd").each { |line|
  puts line
}
