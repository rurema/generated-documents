instance method File::Stat#ftype

ftype -> String[permalink][rdoc][edit]

ファイルのタイプを表す文字列を返します。

文字列は以下のうちのいずれかです。 

"file"
"directory"
"characterSpecial"
"blockSpecial"
"fifo"
"link"
"socket"

"unknown"

例

fs = File::Stat.new($0)
p fs.ftype #=> "file"
p File::Stat.new($:[0]).ftype #=> "directory"

1.8 以降では、属性メソッドがシステムでサポートされていない場合 nil が返ります。なお、1.7 以前では 0 が返っていました。
