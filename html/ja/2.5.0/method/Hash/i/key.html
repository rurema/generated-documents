instance method Hash#index

key(val) -> object[permalink][rdoc][edit]
index(val) -> object

値 val に対応するキーを返します。対応する要素が存在しない時には nil を返します。

該当するキーが複数存在する場合、どのキーを返すかは不定です。

Hash#index は obsolete です。使用すると警告メッセージが表示されます。

[PARAM] val:
探索に用いる値を指定します。 
例

h = {:ab => "some" , :cd => "all" , :ef => "all"}

p h.key("some") #=> :ab
p h.key("all") #=> :cd
p h.key("at") #=> nil
