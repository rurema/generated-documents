instance method OpenSSL::BN#set_bit!

set_bit!(n) -> self[permalink][rdoc][edit]

自身の n ビット目を1にします。 


require 'openssl'

a = OpenSSL::BN.new("128")
a.set_bit!(0)
a # => 129
[PARAM] n:
1にするビットの位置
[EXCEPTION] OpenSSL::BNError:
計算時エラー
