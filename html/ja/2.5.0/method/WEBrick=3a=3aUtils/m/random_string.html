module function WEBrick::Utils.#random_string

random_string(len) -> String[permalink][rdoc][edit]

A-Z, a-z, 0-9 からなる長さ len のランダムな文字列を返します。

[PARAM] len:
得たい文字列の長さを整数で指定します。単位はバイトです。 
require 'webrick'
p WEBrick::Utils.random_string(10)   #=> "xF0jwU4RJd"
