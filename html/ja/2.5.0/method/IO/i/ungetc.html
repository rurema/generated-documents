instance method IO#ungetc

ungetc(char) -> nil[permalink][rdoc][edit]

指定された char を読み戻します。

[PARAM] char:
読み戻したい1文字かそのコードポイントを指定します。
[EXCEPTION] IOError:
読み戻しに失敗した場合に発生します。また、自身が読み込み用にオープンされていない時、自身がまだ一度も read されていない時に発生します。 
f = File.new("testfile")   # => #<File:testfile>
c = f.getc                 # => "い"
f.ungetc(c)                # => nil
f.getc                     # => "い"
