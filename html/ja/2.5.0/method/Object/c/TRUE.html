constant Object::TRUE

TRUE -> TrueClass[permalink][rdoc][edit]

非推奨です。代表的な真の値。true と同じ。

この定数は過去との互換性のために提供されています。擬似変数 true を使ってください。

Ruby では false と nil が偽として扱われます。偽でない値(false でも nil でもない値) は全て真とみなされます。
