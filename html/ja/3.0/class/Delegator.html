class Delegator

[edit]
クラス・モジュールの継承リスト:
  1. BasicObject
  2. Kernel
  3. Object
  4. Delegator

要約

サブクラスにメソッド委譲の仕組みを提供する抽象クラス。

メソッド委譲を行う場合は、本クラスを継承しDelegator#__getobj__を再定義する必要があります。

具体的な使用例については、SimpleDelegatorを参照してください。

インスタンスメソッド

! self -> bool[permalink][rdoc][edit]

自身を否定します。

self != obj -> bool[permalink][rdoc][edit]

自身が与えられたオブジェクトと等しくない場合は、真を返します。そうでない場合は、偽を返します。

[PARAM] obj:
比較対象のオブジェクトを指定します。
self == obj -> bool[permalink][rdoc][edit]

自身が与えられたオブジェクトと等しい場合は、真を返します。そうでない場合は、偽を返します。

[PARAM] obj:
比較対象のオブジェクトを指定します。
__getobj__ -> object[permalink][rdoc][edit]

委譲先のオブジェクトを返します。

本メソッドは、サブクラスで再定義する必要があり、デフォルトでは NotImplementedError が発生します。

[EXCEPTION] NotImplementedError:
サブクラスにて本メソッドが再定義されていない場合に発生します。
__setobj__(obj) -> object[permalink][rdoc][edit]

委譲先のオブジェクトをセットします。

[PARAM] obj:
委譲先のオブジェクトを指定します。
[EXCEPTION] NotImplementedError:
サブクラスにて本メソッドが再定義されていない場合に発生します。
freeze -> self[permalink][rdoc][edit]

自身を凍結します。

[SEE_ALSO] Object#freeze

marshal_dump -> object[permalink][rdoc][edit]

シリアライゼーションをサポートするためにDelegator#__getobj__ が返すオブジェクトを返します。

marshal_load(obj) -> object[permalink][rdoc][edit]

シリアライズされたオブジェクトから、Delegator#__getobj__ が返すオブジェクトを再現します。

[PARAM] obj:

method_missing(m, *args) -> object[permalink][rdoc][edit]

[PARAM] m:
メソッドの名前（シンボル）
[PARAM] args:
メソッドに渡された引数
[RETURN]
委譲先のメソッドからの返り値
