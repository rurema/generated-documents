instance method NilClass#rationalize

rationalize -> Rational[permalink][rdoc][edit]
rationalize(eps) -> Rational

0/1 を返します。

[PARAM] eps:
許容する誤差

引数 eps は常に無視されます。 

例

nil.rationalize      # => (0/1)
nil.rationalize(100) # => (0/1)
nil.rationalize(0.1) # => (0/1)
