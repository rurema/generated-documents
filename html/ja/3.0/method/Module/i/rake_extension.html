instance method Module#rake_extension

rake_extension(method) { ... } -> ()[permalink][rdoc][edit] [added by rake]

与えられたブロック内で既に存在するメソッドを再定義しようとした場合に警告を表示します。この場合、ブロックは評価されません。

[PARAM] method:
ブロック内で再定義する予定のメソッド名を指定します。

例: 

class String
  rake_extension("xyz") do
    def xyz
      ...
    end
  end
end
