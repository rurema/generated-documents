instance method IO#readbyte

readbyte -> Integer[permalink][rdoc][edit]

IO から1バイトを読み込み整数として返します。既に EOF に達していれば EOFError が発生します。

[EXCEPTION] EOFError:
既に EOF に達している場合に発生します。 
例

IO.write("testfile", "123")
File.open("testfile") do |f|
  begin
    f.readbyte  # => 49
    f.readbyte  # => 50
    f.readbyte  # => 51
    f.readbyte  # => 例外発生
  rescue => e
    e.class # => EOFError
  end
end
