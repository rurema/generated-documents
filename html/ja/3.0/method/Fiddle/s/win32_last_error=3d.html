singleton method Fiddle.win32_last_error=

win32_last_error=(errno)[permalink][rdoc][edit]

Fiddle.win32_last_error で返される値を設定します。

errno は fiddle が設定するのでユーザはこのメソッドを使わないでください。

このメソッドは Windows 環境でのみ定義されています。
