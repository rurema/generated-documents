instance method String#b

b -> String[permalink][rdoc][edit]

self の文字エンコーディングを ASCII-8BIT にした文字列の複製を返します。 

例

'abc123'.encoding    # => #<Encoding:UTF-8>
'abc123'.b.encoding  # => #<Encoding:ASCII-8BIT>
