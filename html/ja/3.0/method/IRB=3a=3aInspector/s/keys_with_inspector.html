singleton method IRB::Inspector.keys_with_inspector

keys_with_inspector(inspector) -> Array[permalink][rdoc][edit]

引数で指定した IRB::Inspector に対応する key の配列を返します。

[PARAM] inspector:
IRB::Inspector オブジェクトを指定します。 
IRB::Inspector.keys_with_inspector(IRB::Inspector::INSPECTORS[true])
# => [true, :p, "p", :inspect, "inspect"]
