instance method Time#thursday?

thursday? -> bool[permalink][rdoc][edit]

self の表す時刻が木曜日である場合に true を返します。そうでない場合に false を返します。 


t = Time.local(1995, 12, 21)     # => 1995-12-21 00:00:00 +0900
p t.thursday?                    # => true
