variable $,

$, -> String | nil[permalink][rdoc][edit]

デフォルトの出力フィールド区切り文字列です。 Array#join で引数を省略した場合と、 Kernel.#print の各引数の間で出力されます。

デフォルト値は nil で、空文字列と同じ結果になります。

Ruby 2.7からは nil 以外に変更することは非推奨になったため、 nil 以外を代入すると警告がでるようになりました。

この変数はグローバルスコープです。
