instance method Psych::SyntaxError#file

file -> String|nil[permalink][rdoc][edit]

エラーが生じたファイルの名前を返します。

Psych.load_file で指定したファイルの名前や Psych.load の第2引数で指定した名前が返されます。パース時にファイル名を指定しなかった場合は nil が返されます。
