instance method Gem::Requirement#specific?

specific? -> bool[permalink][rdoc][edit]

条件に上限のある指定で、最新のバージョンにマッチしない可能性のある場合は、true を返します。 


p Gem::Requirement.new(">= 3").specific?  # => false
p Gem::Requirement.new("~> 3").specific?  # => true
p Gem::Requirement.new("=  3").specific?  # => true
