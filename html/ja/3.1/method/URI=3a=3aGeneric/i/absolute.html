instance method URI::Generic#absolute

absolute -> bool[permalink][rdoc][edit]
absolute? -> bool

自身に scheme が設定されているなら true を返します。そうでない場合は false を返します。

例: 

require 'uri'
p URI.parse('http://example.com/').absolute?    #=> true
p URI.parse('./').absolute?                     #=> false
