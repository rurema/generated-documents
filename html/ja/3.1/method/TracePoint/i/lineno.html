instance method TracePoint#lineno

lineno -> Integer[permalink][rdoc][edit]

発生したイベントの行番号を返します。

[EXCEPTION] RuntimeError:
イベントフックの外側で実行した場合に発生します。 
例

def foo(ret)
  ret
end
trace = TracePoint.new(:call, :return) do |tp|
  tp.lineno
end
trace.enable
foo 1
# => 1
# 3
