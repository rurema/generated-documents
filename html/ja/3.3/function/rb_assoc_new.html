function rb_assoc_new

[edit]
VALUE rb_assoc_new(VALUE a, VALUE b)

[a,b] を返します。

対応するRubyコード 

[a, b]

使用例 

VALUE assoc_string(VALUE str_a, VALUE str_b)
{
  Check_Type(str_a, T_STRING);
  Check_Type(str_b, T_STRING);
  return rb_assoc_new(str_a, str_b);
}
