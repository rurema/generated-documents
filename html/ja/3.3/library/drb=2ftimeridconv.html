library drb/timeridconv

[edit]

要約

DRb::DRbIdConv の拡張 DRb::TimerIdConv を定義するライブラリ。 DRb::DRbIdConv の GC 問題をタイムアウトを定めることで部分的に解決します。

クラス

DRb::TimerIdConv

オブジェクトと識別子を相互に変換するクラスです。これによって識別子に変換されたオブジェクトは一定時間 GC から保護されます。
