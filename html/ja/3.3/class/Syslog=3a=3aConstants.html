module Syslog::Constants

[edit]
クラス・モジュールの継承リスト:
  1. Syslog::Macros
  2. Syslog::Level
  3. Syslog::Facility
  4. Syslog::Option
  5. Syslog::Constants

要約

このモジュールにはシステムで使用可能なLOG_*定数、モジュール関数が定義されています。

例: 

require 'syslog'
include Syslog::Constants

それぞれの定数、モジュール関数は以下のモジュールに分けて定義されています。

  • Syslog::Option
  • Syslog::Facility
  • Syslog::Level
  • Syslog::Macros
