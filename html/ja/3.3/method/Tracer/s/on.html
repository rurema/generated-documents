singleton method Tracer.on

on -> nil[permalink][rdoc][edit]
on {...}

トレース出力を開始します。ブロックを与えられた場合はそのブロック内のみトレース出力を行います。 

require 'tracer'

Tracer.on
class Test
  def test
    b = 2
  end
end

t = Test.new
t.test
