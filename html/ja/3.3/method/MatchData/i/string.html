instance method MatchData#string

string -> String[permalink][rdoc][edit]

マッチ対象になった文字列の複製を返します。

返す文字列はフリーズ(Object#freeze)されています。 

例

m = /(.)(.)(\d+)(\d)/.match("THX1138.")
m.string   # => "THX1138."
フィードバックを送る / このマニュアルを編集する
