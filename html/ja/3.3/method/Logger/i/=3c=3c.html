instance method Logger#<<

self << msg -> Integer | nil[permalink][rdoc][edit]

ログを出力します。

[PARAM] msg:
ログに出力するメッセージ。 
例

require 'logger'
logger = Logger.new(STDOUT)

logger << "add message"

# => add message
