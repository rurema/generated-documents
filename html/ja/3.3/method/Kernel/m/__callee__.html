module function Kernel.#__callee__

__callee__ -> Symbol | nil[permalink][rdoc][edit]

現在のメソッド名を返します。メソッドの外で呼ばれると nil を返します。 

例

def foo
  p __callee__
end
alias :bar :foo
foo # => :foo
bar # => :bar
p __callee__ # => nil

Kernel.#__method__ とは異なり、現在のメソッド名が alias されたメソッドの場合には alias 先のメソッド名を返します。
