constant IRB::ExtendCommandBundle::OVERRIDE_PRIVATE_ONLY

OVERRIDE_PRIVATE_ONLY -> 2[permalink][rdoc][edit]

irb でコマンドのエイリアスを定義する際に、既にこれからエイリアス先に指定したメソッド名と同名の public メソッドが定義済みではなかった場合のみ、エイリアスを定義する事を指定するフラグです。
