instance method Pathname#each_filename

each_filename {|v| ... } -> nil[permalink][rdoc][edit]

self のパス名要素毎にブロックを実行します。 

例

require 'pathname'

Pathname.new("/foo/../bar").each_filename {|v| p v}

# => "foo"
#    ".."
#    "bar"
フィードバックを送る / このマニュアルを編集する
