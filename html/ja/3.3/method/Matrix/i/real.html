instance method Matrix#real

real -> Matrix[permalink][rdoc][edit]

行列の実部を返します。 

例

require 'matrix'
Matrix[[Complex(1,2), Complex(0,1), 0], [1, 2, 3]]
#  => 1+2i  i  0
#        1  2  3
Matrix[[Complex(1,2), Complex(0,1), 0], [1, 2, 3]].real
#  =>    1  0  0
#        1  2  3
フィードバックを送る / このマニュアルを編集する
