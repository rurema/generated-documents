instance method Matrix#inspect

inspect -> String[permalink][rdoc][edit]

自分自身を見やすい形式に文字列化し、その文字列を返します。 

例

require 'matrix'
a1 = [1, 2]
a2 = [3, 4.5]
m = Matrix[a1, a2]

p m.inspect # => "Matrix[[1, 2], [3, 4.5]]"
