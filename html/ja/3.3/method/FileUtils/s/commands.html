singleton method FileUtils.commands

commands -> Array[permalink][rdoc][edit]

何らかのオプションを持つメソッド名の配列を返します。 


require 'fileutils'
FileUtils.commands  # => ["chmod", "cp", "cp_r", "install", ...]
