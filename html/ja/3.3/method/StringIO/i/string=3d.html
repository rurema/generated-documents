instance method StringIO#string=

string=(buf)[permalink][rdoc][edit]

自身が表す文字列を指定された buf に変更します。

buf はバッファとして使われ、書き込みメソッドによって書き換えられます。自身は読み書き両用になりますが、 buf がフリーズされている場合には読み取り専用になります。 pos と lineno は 0 にセットされます。

[PARAM] buf:
自身が新たに表す文字列を指定します。
[EXCEPTION] TypeError:
buf が nil の場合に発生します。
